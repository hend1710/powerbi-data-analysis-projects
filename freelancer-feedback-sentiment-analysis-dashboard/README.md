# Freelancer Feedback & Market Insights Dashboard

![Dashboard Preview](dashboard.png)

This project analyzes freelancer survey data to understand global trends in salaries, demographics, and technical skills within the freelance market.

## Tools Used
- Power BI
- Power Query
- DAX
- Data Modeling

## Key Insights
- Salary distribution among freelancers
- Age and demographic patterns
- Most common roles in the data industry
- Demand for programming languages such as Python and SQL
